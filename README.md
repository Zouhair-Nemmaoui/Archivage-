Status: Bewerbungsprojekt für die Ausbildung zum Fachinformatiker für Anwendungsentwicklung 🚀.

Dieses Projekt ist eine Web-Applikation zur effizienten Verwaltung und Archivierung von Radioinhalten. 
Es dient als praktischer Nachweis meiner Programmierkenntnisse in der Full-Stack-Entwicklung 
und zeigt meine Fähigkeit, komplexe Datenbankstrukturen mit modernen Frameworks zu verbinden.


Tech Stack Language : Laravel

Database: MySQL (Relational Database Management)

Frontend: - React.js  - CSS3 

Tools: XAMPP, Bootstrap for Responsive Design
